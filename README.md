# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data using Microsoft Excel to identify trends in revenue, profit, and customer behavior.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Key Features
- Interactive sales dashboard
- Sales analysis by category, state, and month
- Top performing customers
- Profit trends over time
- Dynamic filtering using slicers

## Insights
- Identified top-performing product categories
- Analyzed customer contribution to total sales
- Visualized monthly revenue and profit trends
